pwd stands for "print working directory".
ls stands for "list".
cd stands for "change directory".
You can see what's in a file with more <filename>.


List the contents of folder in "long list format". Do that by adding the -l flag to the "list" command.

You can copy a file with cp <file> <destination>. cp stands for "copy"

You can remove a file with rm <filename>

You can rename them with mv like this: mv <filename> <new_filename>. mv stands for "move", it can rename or move something. Rename roboto.font to roboto.woff


You can use find to find things or view a file tree
